Maintenance Facility System

Aug 2019 - Dec 2019

Description

Led a team of 5 in developing a web application for a maintenance facility system that lets users report issue of an activity center and these reports are assigned by the admin and resolved by the repairers. The web app had the front end of HTML, CSS and back-end of SQL and was tested using Selenium.
